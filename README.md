# Anita Tailoring Studio Website

## Project Overview

Anita Tailoring Studio is a responsive business website created to promote a local lady tailor and showcase tailoring services online.

The website helps customers view services, designs, pricing, location, and book appointments easily.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS Animation Library
- GSAP Animation Library
- Remix Icons

---

## Website Features

- Responsive design for mobile and desktop
- Attractive hero section
- About tailor section
- Services showcase
- Image gallery
- Pricing section
- Customer testimonials
- Google Maps location
- Appointment booking form
- WhatsApp integration
- FAQ section
- Before and After alteration showcase
- Scroll animations

---

## Services Provided

- Blouse Stitching
- Kurti Stitching
- Salwar Suit Stitching
- Designer Dresses
- Clothing Alteration

---

## Project Purpose

This project was developed as part of a Social Internship to support local artisans and small businesses by providing them an online presence.
